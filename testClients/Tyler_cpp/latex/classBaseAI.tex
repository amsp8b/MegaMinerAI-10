\subsection{\-Detailed \-Description}
\-A basic \hyperlink{classAI}{\-A\-I} interface. 

\-This class implements most the code an \hyperlink{classAI}{\-A\-I} would need to interface with the lower-\/level game code. \-A\-Is should extend this class to get a lot of boiler-\/plate code out of the way \-The provided \hyperlink{classAI}{\-A\-I} class does just that.
